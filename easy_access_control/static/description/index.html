Easy Access Control

Control who can create, edit, delete, or view records in Odoo easily and safely.

Overview

Easy Access Control helps administrators manage user-level permissions for different models in Odoo. You can restrict read, create, edit, and delete actions for selected users and improve data safety.

Why this module?

Many Odoo users face accidental data changes by staff or limited-access users. This module protects your data by restricting unwanted actions and giving administrators a simple way to manage access without complex customization.

Key Features

  • User-based access control
  • Model-level permission management
  • Block read access
  • Block create action
  • Block edit action
  • Block delete action
  • Admin and superuser bypass
  • Active / inactive rule support
  • Notes field for internal reference
  • Duplicate rule prevention
  • Improved user-facing warning messages

How It Works

  1. Create a new access rule
  2. Select the user
  3. Select the model
  4. Set Read / Create / Write / Delete permissions
  5. Save and apply instantly

Rule Fields

  • Active: Enable or disable the rule
  • Rule Name: Internal rule name
  • User: User to whom the rule applies
  • Model: Target model
  • Read: Allow or block view access
  • Create: Allow or block create action
  • Write: Allow or block edit action
  • Delete: Allow or block delete action
  • Notes: Internal notes for administrators

Safety Improvements

  • Admin bypass for safer management
  • Read access restriction support
  • Better warning messages for restricted actions
  • SQL constraint to prevent duplicate rules for the same user and model
