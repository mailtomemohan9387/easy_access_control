Easy Access Control

Control who can create, edit, or delete records in Odoo easily and safely.

Overview

Easy Access Control helps administrators manage user-level permissions for different models in Odoo. You can restrict create, edit, and delete actions for selected users and improve data safety.

Key Features

  • User-based access control
  • Model-level permission management
  • Block create action
  • Block edit action
  • Block delete action
  • Simple and easy configuration
  • Improves data safety

How It Works

  1. Create a new access rule
  2. Select the user
  3. Select the model
  4. Set create / write / delete permissions
  5. Save and apply instantly

Use Cases

  • Protect contacts and master data
  • Restrict access for junior users
  • Improve security in multi-user environments
  • Avoid accidental changes or deletions
