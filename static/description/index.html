Easy Access Control

Control who can create, edit, or delete records in Odoo — easily and safely.

Problem it solves

In many Odoo systems, users may accidentally create, edit, or delete important business data. This can lead to data loss, confusion, and operational issues. Easy Access Control helps administrators prevent such problems by controlling user actions.

Key Features

  • User-based access control
  • Model-level permission management
  • Block Create action
  • Block Edit action
  • Block Delete action
  • Simple rule setup and configuration
  • Lightweight and easy to use

How it works

  1. Create an access rule
  2. Select the user
  3. Select the model
  4. Allow or restrict Create, Edit, and Delete actions
  5. Save the rule and apply instantly

Use Cases

  • Prevent deletion of important contacts
  • Restrict junior staff from editing data
  • Allow view-only access for certain users
  • Improve data safety in business operations

Supported Platforms

  • Odoo.sh
  • On-Premise / Self-Hosted Odoo

Important Note

This module is not supported on Odoo Online, because Odoo Online does not allow installation of custom third-party modules.
